# Installation - CoreUI

## Prerequisites
Before you begin, make sure your development environment includes Node.jsÂ® and an npm package manager.

## Node.js
Angular 16 requires Node.js requires Node.js LTS version ^16.14 or ^18.10.

- To check your version, run node -v in a terminal/console window.
- To get Node.js, go to nodejs.org.
  
## Angular CLI

Install the Angular CLI globally using a terminal/console window.

```Bash
npm install -g @angular/cli
```
See also: Angular setup

## Npm
```Bash
npm install @coreui/angular@4
```

## CoreUI CSS files
```Bash
npm install @coreui/coreui
```

## Bootstrap CSS files
```Bash
npm install bootstrap
```
